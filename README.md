This project is a pixel-perfect implementation of the Ellty frontend test assignment.
The goal of this task is to replicate the provided Figma design as accurately as possible,
including sizes, spacing, colors, UI states, and interactions.

# Features
- Pixel-perfect layout based on Figma
- 8 checkbox states (default, hover, active, checked, indeterminate, disabled)
- Button states (default, hover, active, disabled)
- Scrollable page list (6 pages visible via vertical scroll)
- Auto-select logic for "All pages"

# Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
